### Pistas

<div class="hint" title="Haz clic para obtener una pista de estilo de código">

Si comparas dos variables para obtener un resultado `Boolean`, puedes usar la notación corta. 
Por ejemplo, el siguiente código:
```
fun isNotEqual(a: Int, b: Int) = if (a != b) {
    true
} else {
    false
}
```
puede ser reemplazado por este:
```
fun isNotEqual(a: Int, b: Int) = a != b
```
